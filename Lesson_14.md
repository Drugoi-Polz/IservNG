## Основы ООП (Объектно-ориентированное программирование)

### Что такое класс:  

**Класс** — это "чертёж" для создания объектов. Описывает, какие данные (атрибуты) и действия (методы) будут у объекта.

```python
class Dog:
```

### Что такое объект:
**Объект** — это экземпляр класса, созданный по "чертежу"

```python
my_dog = Dog()
```

### Атрибуты:
**Атрибуты** — это переменные внутри класса. Хранят данные об объекте

```python
class Dog:
    def __init__(self, name, age):
        self.name = name   # атрибут
        self.age = age     # атрибут
```

### Методы:
**Методы** — это функции внутри класса. Описывают действия объекта

```python
class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):  # метод
        print("Гав-гав!")
```
#### **Конструктор** __init__ - специальный метод, который вызывается при создании объекта. Используется для настройки начальных значений

```python
def __init__(self, name, age):
    self.name = name
    self.age = age
```

**self** — ссылка на текущий объект. Обязательный первый параметр любого метода

#### Пример полного класса:
```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name} говорит: Гав-гав!")

    def happy_birthday(self):
        self.age += 1
        print(f"С днём рождения, {self.name}! Теперь тебе {self.age} лет")
```
#### Использование класса:
```python
my_dog = Dog("Бобик", 3)
my_dog.bark()            # Бобик говорит: Гав-гав!
my_dog.happy_birthday()  # С днём рождения, Бобик! Теперь тебе 4 лет
```
