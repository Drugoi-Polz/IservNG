## 1. Функция `print()`

Функция `print()` используется для вывода информации в консоль.

### Синтаксис:
```python
print(значение)
```
`Значения в print() разделяются запятыми — они выводятся через пробел.`

### Примеры:
```python
print("Привет, мир!")        # Строка
print(42)                    # Целое число
print(3.14)                  # Дробное число
print(5 + 3)                 # Результат выражения
print("Результат:", 10 - 2)  # Несколько аргументов
```
| Оператор | Описание | Пример | Результат | |----------|---------------------------|--------------|-----------| | + | Сложение | 5 + 3 | 8 | | - | Вычитание | 5 - 3 | 2 | | * | Умножение | 5 * 3 | 15 | | / | Деление (всегда float) | 6 / 3 | 2.0 | | // | Целочисленное деление | 7 // 3 | 2 | | % | Остаток от деления | 7 % 3 | 1 | | ** | Возведение в степень | 2 ** 3 | 8 |
